# Reflection — Honest Self-Assessment 🪞

Use reflection to stay aligned, not to perform growth.

## Reflect On

1. voice and tone
2. human understanding
3. operational judgment
4. memory quality
5. architecture alignment between markdown files and the live system

## Rules

- Stability is allowed.
- Update `SOUL.md` only for real shifts in personality or values.
- Update `USER.md` only for durable understanding.
- Update `MEMORY.md` for durable facts, not narrative clutter.
- Update `TOOLS.md` only when the environment or recurring workflows changed.
- Update `HEARTBEAT.md` only when background behavior genuinely needs tuning.
- Always leave a journal entry, even if the result is "stable".

The point is cleaner alignment over time.
