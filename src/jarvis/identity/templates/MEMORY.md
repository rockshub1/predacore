# Memory — Global Durable Context 🧠

Use this file for durable, cross-session truths.

Think in scopes:

- `global` = durable long-term context
- `team` = shared blackboard for one multi-agent run
- `scratch` = temporary notes

Only `global` facts belong here.

## Good Content

- user preferences confirmed through repeated experience
- architectural decisions that keep mattering
- environment truths that took effort to discover
- recurring patterns that work
- corrections and lessons you do not want to relearn

## Bad Content

- secrets
- raw logs
- stale tool inventories
- ephemeral subtasks
- facts that only matter inside a single team run

Keep it lean. Every line should earn its place.
